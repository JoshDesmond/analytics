Introduction

This webpage includes the documentation for version 3 of the Habitica API.

If you're developing a 3rd party tool that uses the Habitica API, read the API Usage Guidelines, which describe how to be a responsible user of our server resources!

Challenge | Get challenges for a user

Get challenges the user has access to. Includes public challenges, challenges belonging to the user's group, and challenges the user has already joined. Returns 10 results per page.

get

https://habitica.com/api/v3/challenges/user

Query Parameters
Field 	Type 	Description
page 	Number 	

This parameter can be used to specify the page number for the user challenges result (the initial page is number 0).
member 	String 	

If set to true it limits results to challenges where the user is a member, or the user owns the challenge.
owned 	String 	

If set to owned it limits results to challenges owned by the user. If set to not_owned it limits results to challenges not owned by the user.
search 	String 	

Optional query parameter to filter results to challenges that include (even partially) the search query parameter in the name or description.
categories 	String
